# inova-demo

## Description

A Vuetify + Sass starter template including best practices for code styling, testing, and building. 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
